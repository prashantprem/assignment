
# MangoNews

A mangofied News App with clean UI to show list of articles and can open them in web browser :)


## Tech
- Android Studio
- JetPack Compose
- Dependency Injection using Dagger Hilt
- Clean Architecture (MVVM)
- Firebase Cloud Messaging
- Kotlin
